## Quick Start

### Clone repository

```bash
git clone https://github.com/AleksVedenyev/devops-engineer-from-scratch-project-313.git
cd devops-engineer-from-scratch-project-313
```

### Install dependencies

```bash
make install
```

or directly with `uv`:

```bash
uv sync
```

### Run

```bash
make run
```
